HTML
Ctrl + / : 주석이 가능
<h1>~<h6>, <p>, <br>, <hr>, <blockquote>, <cite>, <pre>, <strong>
<h1>~<h6> : 제목 태그
<p> : 문단 태그
<br> : 줄바꿈 태그
<hr> : 구분선 태그
<blockquote cite="https://www."> : 인용 태그 / cite : 출처를 정의하는 속성
<cite> : 작품의 제목 등을 정의하는 요소
<pre> : 그대로 구현되는 태그
<strong> : 글씨 굵게
